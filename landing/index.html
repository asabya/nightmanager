The Nightmanager orchestrates your Pi delegation.

The Nightmanager turns Pi into a multi-specialist team: Finder for codebase search, Oracle for debugging, Worker for implementation, and Manager for orchestration. Install once, delegate intelligently.

Features

Everything you need for intelligent coding delegation

🔍
Finder

Codebase search and exploration. Locate features, trace module connections, and map codebase flows.

🔮
Oracle

Reasoning and debugging. Root-cause analysis, trade-off planning, and deciding the best next probe.

⚡
Worker

Implementation with minimal diffs. Focused edits, smallest viable fix, and verified changes.

🎯
Manager

Orchestration across specialists. Coordinates finder, oracle, and worker workflows.

Tools

Each tool has a focused role and best-use case

finder
Codebase Search

Locate where features are implemented, trace how modules connect, and explore codebase structure.

Best for: "Use finder to find where authentication is handled"
oracle
Reasoning & Debugging

Debug ambiguous failures, root-cause analysis, and trade-off-aware architectural planning.

Best for: "Use oracle to debug why auth fails intermittently"
worker
Implementation

Smallest viable changes with focused execution. Minimal diffs with concrete verification.

Best for: "Use worker to make the smallest possible fix"
manager
Orchestration

Coordinate multi-phase workflows across specialists. Smart delegation with handoff artifacts.

Best for: "Use manager to investigate, choose safest fix, implement, verify"

Nightmanager Workflow

Autonomous TODO implementation with Manager. Delegate complex tasks and let nightmanager work while you sleep. Manager orchestrates finder → oracle → worker with full auditability through handoff artifacts.
